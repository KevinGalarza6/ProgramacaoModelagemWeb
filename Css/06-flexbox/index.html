    6 - Flexbox

    Objetivo: Utilizar o Flexbox para criar layouts flexíveis e responsivos. Conteúdo: Conceito de Flexbox. Container flex e itens flex. Propriedades do container: flex-direction, justify-content, align-items, flex-wrap. Propriedades dos itens: flex-grow, flex-shrink, flex-basis, align-self. Prática: Criar um layout de galeria de imagens utilizando Flexbox.
